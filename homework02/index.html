Orientações

Fazer as alterações necessárias para que, ao alterar qualquer opção do "customizador", as alterações sejam visíveis em tela, porém, as mesmas devem ser guardadas na session storage, caso o usuário decida salvar a alteração, as mesmas devem ser guardadas no Local Storage

Layout livre, fiquem a vontade pra brincar com o CSS e com o javascript ;)

Alterações mínimas:

  • Fonte
  • Background - Página
  • imagem do banner
  • alinhamento do texto dos parágrafos
  • cor dos links (ou botoes) do menu
